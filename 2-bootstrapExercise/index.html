Bootstrap is an open source toolkit for developing with HTML, CSS, and JS. Quickly prototype ideas or build an entire app with Sass variables, mixins, responsive grid system, extensive prebuilt components, and powerful plugins built on jQuery.

Bootstrap is a great tool for building and editing websites. Its mobile first layout and classes for easy editing make building and designing a breeze. Everything is responsive with bootstrap, your content will stretch and shrink with different screen sizes to always fit and look great. you can also add expands and breakpoints to make your content chnage when you want it to instead of automatically.
Bootstrap has several different base colours to apply to content, and even more you can use with Sass variables. These paragraphs are styled with some of the base colours, the first one is info, second is warning, this one is danger, and the last one is success. You can add these colours to content by adding the colour name class to it, for example this text is coloured by the class "text-danger", while the buttons below are with "btn-colour".
Bootstrap has an extremly powerful flexible grid system, its easy to use and makes responsive, complex layouts that look great. To get started create a div with the class of "row", then all other divs inside that first one will have a class of "col". Another way to do it is to create a div with a class of "row row-cols-" after that final dash insert the number of columns you wish to have in yout layout. All content within that div will organize itself.

You can even reorder items in the grid to easily see what different styles and layouts would look like. Like everything in bootstrap, this is done with a series of classes. So if you wanted to move an item to the end of the grid, you could add a class of "order-last" to that item.

Bootstrap also has a wide variety of cards availible to use. This is an image card, it could be used for showcasing locations or even personal Photography and art. Its a great addition for Portfolio websites.

Bootstrap can even add paginations to your website, those are the little navs at the bottom of the screen, typically seen in web browers like Google. Paginations can make navigating a site simple, especially if the site has a lot of content to scroll through. Since there aren't any other pages on this site, this one simply returns you to the top of the screen.
Its easy to get started with Bootstrap, just head over to getbootstrap.com and copy the starter template. Then head over to VS code and copy that into an empty index.html file, thats it. Just two simple steps and your ready to start building creative, professional websites useing bootstrap.
